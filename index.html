What is Django?

What is Django?

  • High-level Python web-framework
  • Lets you create web applications quickly and with less code
  • Models - Object-relation mapping to interact with a database
  • Templates - Django templating engine to render HTML
  • Views - Python functions for request in / request out
  • URL patterns - Regular expressions to define paths

What is a web-framework?

  • Lets you build "web-enabled" applications
  • Provides the groundwork for common stuff every application needs
  • You need to write the code in a "particular manner" a.k.a the "framework"
  • Makes your life easier - lets you focus on the app

How do web applications work?

  1. You open up a browser, enter a URL
  2. Your browser figures out which "host" it should send the request to
  3. The browser actually requests the "resource" you have requested
  4. The web application on the host receives the request
  5. It sends back whatever you have requested (if possible)

Install virtualenvwrapper

  1. Install virtualenvwrapper with pip
    2. $ sudo pip install virtualenvwrapper
  2. Add it to your shell startup
    3. export WORKON_HOME=$HOME/.virtualenvs
source /usr/local/bin/virtualenvwrapper.sh
  3. Reload your sehll startup file
    4. $ source ~/.bashrc

Bootstrap Django

  1. Create a new virtualenv
    2. $ mkvirtualenv tango
  2. Install Django
    3. $ pip install django
  3. Create a new project
    4. $ django-admin.py startproject tango

Create an app

  1. Create a new app
    2. $ python manage.py startapp blog
